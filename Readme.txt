
https://github.com/TextScript-samples/
は、TextScriptで作成したサンプル集です。
作者ごとにリポジトリを持って、サンプルを登録していきます。
作られたサンプルは、自由に持っていって自分のサンプルとして拡張していくことが可能です。
ただし、他の作者のリポジトリ内で変更することは出来ません。
必ず自分のリポジトリ内に持っていって変更して下さい。


興味のある方は、連絡下さい。

TextScriptは、テキストアプリを簡単に作成するためのスクリプト言語です。
スクリプトは、テキストファイルで作成します。
テキストアプリには、テキストボックスを配置したり、背景を指定することができます。
これらをレイアウトと呼び、テキストファイルで作成します。
